=> Click the play button to play
=> Select topic from the given topics
=> Select difficulty level
=> You will be asked some questions based on the selected topic and difficulty
=> Try to answer them correctly by writing the answer in the text box before the timer runs out
=> If you are not able to answer a question correctly or time's up,
    you will be provided with the correct answer, then click anywhere on the screen to continue
=> You can click the exit button to quit
=> In the result page that show you can see your scores, percentage score and detail of each question.
=> Use the arrow keys or the arrow buttons at the bottom of the page(if available)
    to navigate between result pages
=> Click anywhere on the screen to leave the result page and go back to the menu screen
