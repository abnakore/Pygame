# Are You A Math Whiz?
A math tutor game that help kids master their simple Arithmetics

## Setup Instructions

### Install Required Python Modules

```bash
pip install -r requirements.txt
```
### Run the program
##### Note: The program is written with Python 3.10.7

```bash
python main.py
```
